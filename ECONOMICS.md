# Unit Economics

## 1. Cost per Audit
- **Frontend / Edge (Vercel):** $0.000 (Free tier covers 100k+ requests)
- **Database Read/Write (MongoDB + D1):** ~$0.0001 per operation
- **LLM Summary (Claude 3.5 Sonnet):** 
  - Input: ~400 tokens ($0.0012)
  - Output: ~100 tokens ($0.0015)
- **Transactional Email (Resend):** $0.001 per email
- **Total COGS per Audit:** **~$0.003**

## 2. Customer Value
- **Lead Capture Rate:** Assuming a 15% email conversion rate for users who see >$100 in savings.
- **Cost per Lead (CPL):** $0.003 / 0.15 = **$0.02 per lead**.
- **Enterprise Conversion:** If Techvruk closes 1 in 100 high-intent leads for a $5,000 consulting contract, the LTV is massive compared to the $2 cost to acquire those 100 leads.

**Conclusion:** The economics are extremely favorable. The tool can be run indefinitely on free/low-cost tiers while acting as a highly profitable lead generation engine.
